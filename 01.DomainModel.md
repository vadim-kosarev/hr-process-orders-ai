Сгенери классы для доменной модели, которая описывает систему работы с заказами.
Java 21, Spring Boot, JPA, Lombok, DDD (Domain-Driven Design).
Использовать Domain Objects, Value Objects. Обеспечить тестируемость модели независимо.

Order
 - OrderID (UUID)
 - OrderItems (List<OrderItem>)
 - OrderStatus (enum:NEW, IN_PROGRESS, READY, CANCELLED, FAILED)

OrderItem
- productID (UUID)
- quantity (Qty)
- unitPrice (Money)

Генерить только классы для доменной модели, без репозиториев, сервисов и контроллеров.
Без тестов - тесты запрошу отдельно.
